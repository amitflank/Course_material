TFL Tech Certificates

All certificates under “Computer Basics” are required before moving on,
but students may test out of them if they’re already proficient.
After that, they choose at least 2 elective certificates.

Computer Basics

Digital Literacy and Basic Computer Skills

Basic computer-related skills that any adult should know. Learn to identify and use your OS, how to navigate folders and files, how to use keyboard shortcuts to become more efficient, and how to stay safe on the Internet.

Typing

Learn to efficiently use a keyboard, an extremely useful skill regardless of which career you’re pursuing. Learn how to type without looking at the keyboard, how to use the correct finger placement, and how to avoid common mistakes.

Microsoft Word and Google Docs

Learn how to use a word processor. Learn more time-saving keyboard shortcuts and learn how to use various settings to your advantage.

Microsoft Excel and Google Sheets

Learn how to use spreadsheet software. Learn how formulas work and how to organize data for personal and professional use.

Microsoft PowerPoint and Google Slides

Learn how to create and use slideshow presentation software, as well as practice presentation skills.

Programming

Programming I - Fundamental Concepts

Learn fundamental programming concepts, using either Python or Java as determined on a per-student basis. Learn basic computer programming concepts. Understand the difference between various data types, write basic functions, and learn how to use loops. Complete the certificate by writing a simple text-based adventure game.

Programming II - Object-Oriented

Dive deeper into programming concepts with an emphasis on object-oriented programming, using either Python or Java. Review and build upon principles learned in Programming I. Learn about classes and inheritance. Introduction to basic graphics. Utilize all these to create a simple graphics-based game.

Programming III - Algorithms and Problem-Solving

Less about the syntax of any particular programming language and more about general problem-solving with programming. Become familiar with sorting algorithms and focus on making programs more efficient. Learn about “Big O” and how programmers use computers to make our lives better.

Programming IV - C++

After becoming familiar with Python or Java, students may dive into C++, a less beginner-friendly but more powerful programming language. This course will cover the fundamentals of C++ programming, including syntax, data types, control structures, functions, arrays, and pointers. Emphasis on memory management.

Information Technology

Introduction to Linux

Learn about the Linux operating system and its various distributions. You'll explore the command-line interface, learn how to navigate the file system, and perform basic system administration tasks. You'll also learn about the Linux philosophy and the open-source community.

Web Development and Web Design

Intro to Web Development

In this course, you'll learn the basics of web development, including HTML, CSS, and JavaScript. You'll create simple web pages, style them with CSS, and add interactivity with JavaScript. You'll also learn about web standards, accessibility, and responsive design.

Game Development

Game Dev I - The Basics

This course introduces you to the fundamentals of game development. You'll learn about game engines, game mechanics, and game design principles. You'll also learn how to create a simple 2D platform game using either GameMaker Studio 2 or Unity.

Game Dev II - Ludology

Dive deeper into game development concepts and learn about advanced game mechanics, artificial intelligence, and multiplayer networking. Create a more complex top-down adventure game using GMS2 or Unity.

Digital Art

2D Art - GIMP

Use GIMP to create and edit digital images. Learn about the different tools and features of GIMP, and how to use them to create and manipulate images.

3D Art - Blender

Use Blender to create and edit 3D models, animations, and visual effects. Learn about the different tools and features of Blender, and how to use them to create and manipulate 3D objects. Learn about lighting, texturing, and rendering techniques.

3D Printing

Use software to create 3D models, and print it using the 3D printer. Display an understanding of 3D printer hardware and maintenance.
