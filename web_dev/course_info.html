Intro to Web Development (HTML & CSS)

This is a beginner-level course for students interested in web design or web development. The course covers the fundamental principles of frontend web design, providing hands-on experience with HTML and CSS. Students will be introduced to essential concepts like Internet infrastructure, web page creation and publishing, layout techniques, and accessibility. The following link explains the difference between “web design” and “web development,” so students may want to read the article and decide if this course is a good fit for them Web Designer vs Web Developer: What's the Difference?
